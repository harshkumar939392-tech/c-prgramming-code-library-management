# 📚 Library Book Management System

A simple Library Book Management System developed using **C programming**. This project allows users to manage library books by performing basic operations such as adding, viewing, searching, issuing, and returning books.

---

## 🚀 Features

- 📖 Add new books
- 📚 View all books
- 🔍 Search books by ID or title
- 📤 Issue books
- 📥 Return books
- ❌ Delete book records
- ✏️ Update book information
- 💾 File handling for data storage

---

## ▶️ How to Run

### Compile the program

```bash
gcc library.c -o library
```

### Run the program

```bash
./library
```

For Windows (MinGW):

```bash
gcc library.c -o library.exe
library.exe
```
